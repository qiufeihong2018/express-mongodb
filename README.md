## 项目运行

```
项目运行之前，请确保系统已经安装以下应用
1、node (6.0 及以上版本)
2、mongodb (开启状态)
3、GraphicsMagick (裁切图片)
```

```
git clone git@github.com:qiufeihong2018/express-mongodb-tack-out.git  

cd express-mongodb-tack-out

npm install 或 yarn(推荐)

npm run dev

访问: http://localhost:7979（如果已启动前台程序，则不需打开此地址）

```
